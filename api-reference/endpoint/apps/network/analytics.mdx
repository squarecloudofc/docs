---
title: "Custom Domain Analytics"
sidebarTitle: "[...]/network/analytics"
description: "Get analytics for a custom domain. It has a rate limit of 10 request every 5 seconds."
api: "GET https://api.squarecloud.app/v2/apps/{app_id}/network/analytics"
authMethod: "none"
---

<Info>**Is necessary to have at least the [Standard plan](https://squarecloud.app/plans) to use this route.**</Info>

<ParamField header="Authorization" type="string" placeholder="API KEY" required>
  The API key for your account. You can find this in your [account settings](https://squarecloud.app/account).
</ParamField>

<ParamField path="app_id" type="string" placeholder="Application ID" required>
  The ID of the application. You can find this in the URL of your application's dashboard.
</ParamField>

<ParamField query="range" type="string" placeholder="Time range">
  Time range for the analytics data. It can be `daily` or `weekly`. Default is `daily`.
</ParamField>

### Response

<Info>

<ResponseField name="status" type="string">
  Indicates whether the call was successful. `success` if successful, `error` if not.
</ResponseField>
