# 📜 Sobre
Square Cloud é um projeto que nasceu com o intuito de oferecer serviço de hospedagem de forma facilitada, gratuita e sem exigir conhecimentos avançados.
Nossa infraestrutura é baseada em nuvem em contêineres flexíveis, com alta escalabilidade e alto controle relacionado a sua aplicação, você tem acesso instantâneo ao status de sua aplicação.
Contamos com colaboração de doadores para mantermos uma qualidade excepcional e os nossos ideais, que se baseiam em oferecer uma hospedagem gratuita, de alta qualidade. 
Não possuímos nenhum tipo de vínculo com a Discord Inc.
